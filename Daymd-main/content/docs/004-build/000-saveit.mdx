<Warn>在编辑器预览时往往运行较迟钝，但是打包后就不会有这个问题，这是正常现象。</Warn>

当你在 StackBlitz 中或者在本地 Visual Studio Code 完成你的项目的编辑后，你可以按住 `ctrl` `s` 或者 `⌘` `s` 来保存项目。然后，你就可以将代码提交回 GitHub 了。  

## StackBlitz

在项目左上角点击 `commit` 按钮按照提示点击 `commit` 。现在，你更新的代码就保存到 GitHub 了。  

![1](https://jetzihan-img.oss-cn-beijing.aliyuncs.com/blog/1661912742628.png)

出现最近更新的时间。表明提交成功：  

![2](https://jetzihan-img.oss-cn-beijing.aliyuncs.com/blog/1661912963333.png)

## 本地提交

本地提交不再赘述，通过 Git 工具提交即可。
