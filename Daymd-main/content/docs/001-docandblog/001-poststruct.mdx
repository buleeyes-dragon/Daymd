在本节中，我们将阐述 Daymd 中博客的结构。以便让你清楚创建什么样的结构能够在网站上展示什么样的结果。

## 博客的基本结构

博客与文档不同，博客是每一篇独立存在的，不存在集合关系，因此你只能创建单个博客，而不能创建**集合**，正如下面的博客页面所示：  

![1](https://jetzihan-img.oss-cn-beijing.aliyuncs.com/blog/20220830102543.png)

左侧是博客列表，它按照从新到旧的顺序展示。点击即可进入相应的博客页面。在文件结构中，所有的博客均放在 `content/posts` 文件夹下，除博客首页文件 `index.mdx` 外，每篇博客都有单独的文件夹，以及这篇博客的页面文件 `index.mdx` 。  

## 创建博客

现在，我们在 posts 文件夹上右键创建新的文件夹。  

![1](https://jetzihan-img.oss-cn-beijing.aliyuncs.com/blog/20220830121504.png)

<Note title="命名规则">
在为博客文件夹命名时，请按照 `yyyy-MM-dd-blogName` 的格式进行命名。Daymd会按照时间为你排序。  
</Note>

将其命名为 `2022-8-30-myfirstblog`  然后在这个文件夹中添加页面 `index.mdx` 。为这个文件添加内容。  

![1](https://jetzihan-img.oss-cn-beijing.aliyuncs.com/blog/1661842553336.png)

```md
---
title: 博客结构
excerpt: 博客的结构
date: 2022-08-29
backpic: null
---
## 一个博客标题
这是这个标题下的内容。
```

按下 `ctrl`+`s` 保存文件，可以看到右边的执行窗口开始刷新，等待刷新完成。  
刷新完成后，如下：

![1](https://jetzihan-img.oss-cn-beijing.aliyuncs.com/blog/20220830145113.png)

你的博客区成功创建了一片新的博客。但是请注意，**不要**在每篇博客文件夹中再创建其他 `markdown` 文件，否则会引发错误。  
恭喜！现在你按照上面的步骤就可以多篇自己的博客了。  

## 编写一篇博客

像文档一样，我们现在来编写一篇博客。同样地，它也由**格式头**和**内容**组成。博客的格式头相较于文档增加了 `backpic` 字段，它用来在首页展示博客列表时显示背景图片。  

### 无背景模式

在无背景模式下将 `backpic` 字段填写为 `null`。  

```md
---
title: 博客标题
excerpt: 博客的描述
date: 2022-08-29 # 博客创建时间 
backpic: null # 博客列表背景图片
---
```

### 有背景模式

在有背景模式下将 `backpic` 字段填写为图床链接。在后面的章节我们将介绍图床的使用。  

```md
---
title: 博客标题
excerpt: 博客的描述
date: 2022-08-29 # 博客创建时间 
backpic: url(https://api.maho.cc/random-img/pc.php) # 博客列表背景图片
---
```

<Warn>
请注意，背景图片的格式为 `backpic:url(链接)`，而不是 `backpic:链接`。`url` 以及后面的括号是必须的。否则不会生效。
</Warn>

### 例子

下面是一个较完整的博客例子，其标题是 Java , 描述是 Java Note, 创建日期是 2020-8-25，背景图链接为 `https://jetzihan-img.oss-cn-beijing.aliyuncs.com/blog/NavBanner.png` 主要内容是从 `# Java` 开始的下面那些内容：

## Java
Java 是由 Sun Microsystems 公司于 1995 年 5 月推出的高级程序设计语言。

## Java Examples

以上我们使用了两个命令 javac 和 java。
javac 后面跟着的是java文件的文件名，例如 HelloWorld.java。 该命令用于将 java 源文件编译为 class 字节码文件，如： javac HelloWorld.java。
运行javac命令后，如果成功编译没有错误的话，会出现一个 HelloWorld.class 的文件。
java 后面跟着的是java文件中的类名,例如 HelloWorld 就是类名，如: java HelloWorld。
