## 文字加粗

你可以使用`**` 来指定文字加粗。使用方法是使用 `**` 来包裹一个你想加粗的文本。

```md
**这是加粗的文本**。
```

显示效果：

**这是加粗的文本**。

## 文字倾斜

你可以使用`*` 来指定文字倾斜。使用方法是使用 `*` 来包裹一个你想倾斜的文本。

```md
*这是倾斜的文本*
```

显示效果：  

*这是倾斜的文本*

## 同时使用粗体和斜体

你可以同时使用 `*` `**` 来指定文字倾斜。使用方法是使用 `***` 来包裹一个你想倾斜的文本。

```md
***这是倾斜且加粗的文本***
```

显示效果：  

***这是倾斜且加粗的文本***

## 换行

请在每行文字的末尾输入两个**空格**  进行换行。  

```md
这是一行文字 #这一行的末尾没有两个空格
这时下一行文字
```

**显示效果：**  

这是一行文字
这时下一行文字  （并没有换行）

```md
这是一行文字  #这一行的末尾有两个空格
这时下一行文字
```

**显示效果：**  

这是一行文字  
这时下一行文字

## 引用

使用 `>` 就可以指定文字为引用。

```md
> 这是一个引用
```

**显示效果：**  
> 这是一个引用

在引用中可以叠加其他的语法。  

```md
> ### 在应用中的标题
> 在引用中的**加粗**
```

**显示效果：**  

> ### 在引用中的标题

> 在引用中的**加粗**

## 列表

```md
1. First item
2. Second item
3. Third item
4. Fourth item
```

显示效果：  

1. First item
2. Second item
3. Third item
4. Fourth item

也可以使用 `-` 来指定列表。

```md
- First item
- Second item
- Third item
- Fourth item
```

为了避免 `1.` 等格式被渲染为小圆点，可以略去 `1.` 后面的空格，但是记得在每行后面添加两个空格换行。  

```md
1.First item
2.Second item
3.Third item
4.Fourth item
```

你将看到被原封不动渲染出来：  

1.First item  
2.Second item  
3.Third item  
4.Fourth item  

## 小代码块

你可以使用 **``** 来指定一段小代码。这之中的内容不会被解析成 Markdown元素。  

```md
`<div>Hello World</div>`
```

**显示效果**：  

`<div>Hello World</div>`

如果你在你的文章中使用 HTML 语法却不使用 `` 进行包裹，会发生报错。  

## 分割线

要创建分隔线，请在单独一行上使用三个或多个星号 `***`、破折号 `---` 或下划线 `___` ，并且不能包含其他内容。  

```md
***

---

_________________
```

显示效果：  

***

---

_________________
