我们为您提供了 29 种主题，您可以在下面进行点击体验，选择您心怡的主题后，将其**名称变为小写字母**添加到 `daymd.config.js` 中的 `themeSelect` 选项。

不同的主题将影响：

- 字体
- 配色
- UI 样式（卡片、按钮等）
