经过一些坎坷，我们很快就让我们的网站部署上线了，当然，前面都是一些基础的步骤， Daymd 作为一个 Next.js 应用，它有着很强的可拓展性，你可以在 Daymd 的基础上自己创建页面以及其他功能。  
现在，我们来看看 Daymd 的进阶指南。  

## 需要的基础

这一部分需要涉及到前端开发技术。你可能需要掌握基本的 html 、 css 、JavaScript 等技术，在创建页面时，Daymd 要求你创建的是 React 页面，因此要求你掌握 React 基础。

## 页面创建

这一章节会帮助你创建一个自定义的页面，并且能够通过路由访问，在 `daymd.config.js` 中的 `navItems` 配置可以路由。

## 基础样式

这一章节会带你了解可以通过全局 css 定义的样式如何进行修改，从而进一步对页面进行自定义。
