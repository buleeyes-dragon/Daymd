我们为您提供了**告示条**语法，你可以在您的项目中使用它。

## 提示

### 有标题提示

```md
<Tips title="标题">这里面填写内容。</Tips>
```

<Tips title="标题">这里面填写内容。</Tips>

### 无标题提示

```md
<Tips>这里面填写内容。</Tips>
```

<Tips>这里面填写内容。</Tips>
