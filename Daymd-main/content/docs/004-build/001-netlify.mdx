## 介绍

Netlify 是一个提供网站部署服务的开源平台，可以让开发者在线部署静态网站。仅仅需要几部简单的操作就可以完成一个网站的部署与发布。  

## 登录

进入 <To to="https://www.netlify.com/">Netlify</To> 网站，点击右上角 `Sign Up` 进行登录。  

![1](https://jetzihan-img.oss-cn-beijing.aliyuncs.com/blog/1661913403135.png)

<Warn>注意，必须使用 GitHub 进行登录。</Warn>

## 导入仓库

注册完成后，进入控制台界面，选择 `Add a new site`，创建一个新的项目。  

![1](https://jetzihan-img.oss-cn-beijing.aliyuncs.com/blog/1661913530253.png)

选择 `Import an existing project from a Git repository` 然后选择 GitHub 导入：  

![2](https://jetzihan-img.oss-cn-beijing.aliyuncs.com/blog/1661913574817.png)

选择目标仓库（你 Fork 的仓库）。  

![2](https://jetzihan-img.oss-cn-beijing.aliyuncs.com/blog/1661913643644.png)

## 部署

选择部署分支为 `main` 取决于你存放的分支。  

![2](https://jetzihan-img.oss-cn-beijing.aliyuncs.com/blog/20220831104147.png)

下面的选项是部署指令，不用理会。它已经帮我们选好了。  

点击 `Deploy Site` 即可完成搭建。  

点击进入可以看到部署过程。  

![1](https://jetzihan-img.oss-cn-beijing.aliyuncs.com/blog/1661913831131.png)

出现 `Site is live` 即部署完成。  

![2](https://jetzihan-img.oss-cn-beijing.aliyuncs.com/blog/1661913972393.png)

之后，你对你的博客进行修改，增加文章等操作，一旦 GitHub 内容发生变化， Netlify 会自动帮你进行重新部署，待一段时间后即可重新看到。
